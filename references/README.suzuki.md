塩基組成に基づいたプラスミドの宿主域の予測
◯鈴木治夫（慶應義塾大学先端生命科学研究所）

細菌の薬剤耐性や病原性の獲得には、細菌間を移動する染色体外DNA（プラスミド）が関与する。細菌進化におけるプラスミドの重要性を理解するためには、プラスミドの宿主域を知る必要がある。プラスミドは、既知の宿主の染色体と塩基組成が類似しているので、過去に滞在した宿主の塩基組成を獲得したと考えられる。狭宿主域プラスミドは一部の細菌にだけ塩基組成が類似していたのに対し、広宿主域プラスミドは多様な細菌に塩基組成が類似していた。このことは、塩基組成からプラスミドの宿主域を予測できることを示唆する。

抗生物質耐性プラスミドの乱交度を予測

プラスミドは、抗生物質耐性や病原性を細菌集団に広げる。本研究は、プラスミドが滞在した細菌の多様性をDNA配列の情報から推定できることを示した。ゲノム塩基組成 genomic signature を調べると、狭宿主域プラスミドは一部の細菌種の染色体にだけ類似しているのに対し、（多様なプロテオバクテリアで複製する）広宿主域プラスミドは、多様な細菌種の染色体に類似しているか、あるいは（現在までに配列決定されている）全ての細菌の染色体と異なっている。このようにプラスミドのDNA配列は、プラスミドが以前住んでいた場所（宿主細菌）を伝える歴史書とみなせる。ゲノム塩基組成 genomic signature 解析は、水平伝播遺伝子プールの可動性遺伝因子（抗生物質耐性や病原性を有するプラスミド、ウイルス、ゲノムアイランド）の乱交度や潜在的宿主に関する洞察を提供する。

ゲノム塩基組成に基づいたプラスミドの宿主域の予測

プラスミドは細菌の進化に大きく関与しているが、プラスミドが進化してきた宿主域 evolutionary host range に関する知見はほとんど得られていない。プラスミドは、既知宿主染色体と塩基組成 genomic signature が類似していることから、宿主の塩基組成を獲得してきたと考えられる。そこで、プラスミドと細菌817菌株の3連続塩基組成（AAA, AAC, AAG, AAT, ... , TTT の64変数）の比較によりプラスミドの宿主を推定し、分類群と遺伝距離を用いて推定宿主の多様性を評価した。様々な不和合性群 incompatibility (Inc) group の進化的な宿主域を推定したところ、IncPは広宿主域、IncFとIncIは狭宿主域、IncHとIncNはその中間の宿主域を示し、この結果は経験的な宿主域とも一致した。IncA/C、IncP、IncQ、IncU、IncW群に属するプラスミドの中には、何れの細菌とも塩基組成が類似していないものが認められた。この結果は、宿主の配列が未決定か、（様々な細菌を宿主とする）乱交により宿主の塩基組成を獲得しなかった等の仮説を提起する。本法は、ハイスループットゲノム配列決定により大量に得られる新規可動性遺伝因子の宿主域の予測に有効である。

マハラノビス距離を用いた細菌プラスミド・染色体間の塩基組成の比較

可動性遺伝因子のプラスミドは、薬剤耐性や病原性などの機能を宿主細菌に付与する。細菌間の遺伝子水平伝播におけるプラスミドの重要性を理解するためには、プラスミドが進化してきた宿主細菌を知る必要がある。外来性DNAが長期間滞在した宿主の塩基組成を獲得するという仮説（[Amelioration](http://www.ncbi.nlm.nih.gov/pubmed/9089078)）によれば、プラスミドと主染色体の塩基組成比較によってプラスミドの長期滞在宿主を推定できるはずである。細菌の染色体とプラスミドとの間の2連続塩基組成（[genomic signature](https://www.ncbi.nlm.nih.gov/pubmed/10430917); AA, AC, AG, AT, CA, CC, CG, CT, GA, GC, GG, GT, TA, TC, TG, TTの16変数）の非類似度を測る距離尺度として、1/16マンハッタン距離（δ距離）が用いられてきた。本研究では、プラスミドの既知宿主の予測精度は、δ距離よりも（データの分散共分散を考慮する）マハラノビス距離の方が高いことを示した。マハラノビス距離を用いて、炭疽菌 Bacillus anthracis の病原性プラスミドpXO1、腸管出血性大腸菌 Escherichia coli O157:H7 の病原性プラスミドpO157、および宿主未知の広宿主域・多剤耐性プラスミドpB10の宿主を推定した。マハラノビス距離を用いたオリゴヌクレオチド解析は、可動性遺伝因子（プラスミド、ファージ、トランスポゾンなど）の宿主の予測、最適なベクターと宿主の組合わせの決定、および外来性のゲノム領域 genomic island の検出に応用できる。
